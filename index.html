🍏 Kick Clips Viewer

Kick Clips Viewer is a modern Chrome extension that allows you to browse every clip from any Kick.com channel using a fast and convenient interface. Search, sort, watch clips in a built-in player and download them in the highest available quality.

Install from Chrome Web Store View Source on GitHub

✨ Features

🚀 Load Every Clip

Automatically loads all available clips from a Kick channel using the public Kick API.

🔍 Smart Search

Instantly search by clip title, category or creator.

🗂️ Advanced Sorting

Newest, oldest, most viewed or least viewed.

🎬 Built-in Player

Watch clips directly inside the extension with selectable video quality.

📥 Downloads

Download clips in the highest available quality with one click.

🌐 Multi-language

Supports English and Russian with an integrated Settings page.

🛡️ Privacy First

Analytics are disabled by default and only enabled after explicit consent.

⚙️ Open Source

Complete source code is available on GitHub under the MIT License.

How it works

Install the extension from the Chrome Web Store, enter any Kick channel name, and Kick Clips Viewer will automatically fetch every available clip. Browse thousands of clips using pagination, search, sorting, and the integrated video player.
